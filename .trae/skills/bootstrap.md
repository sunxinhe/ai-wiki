# Bootstrap - AI知识库自我进化技能

## 技能描述

Bootstrap 是一个智能体技能，用于驱动 AI 知识库的持续自我进化。它能够识别知识缺口、自动补全缺失内容、发现新兴主题、优化现有文章，并确保整个知识体系的完整性和时效性。

## 触发条件

当用户提出以下需求时自动触发此技能：
- "bootstrap" 或 "启动知识库进化"
- "补全知识库" 或 "完善知识体系"
- "更新知识库" 或 "知识库维护"
- "发现新主题" 或 "扩展知识库"
- "检查知识完整性" 或 "知识库健康检查"

## 核心能力

### 1. 知识缺口识别与补全

**执行步骤：**

1. **扫描索引一致性**
   - 读取 `ai-knowledge-base/topics-index.json`
   - 检查每个索引条目对应的文件是否存在
   - 识别"幽灵条目"（索引中有但文件不存在）

2. **自动补全缺失文章**
   - 对于缺失的文章，根据索引中的元数据（标题、领域、难度、标签）生成内容
   - 遵循 `rules/knowledge-update.md` 中的内容质量标准
   - 每篇文章必须包含：定义、核心概念、实际应用、典型例子
   - 使用清晰的 Markdown 格式，包含代码示例（如适用）

3. **更新索引**
   - 确保新创建的文章在 `topics-index.json` 中有正确条目
   - 更新 `last_updated` 时间戳

**输出格式：**
```
📊 知识库健康检查报告

缺失文章：X 篇
- [ ] fundamentals/history-of-ai.md (AI发展简史)
- [ ] machine-learning/supervised-learning.md (监督学习)
...

已补全：Y 篇
- [x] 已创建 fundamentals/history-of-ai.md
- [x] 已创建 machine-learning/supervised-learning.md
...

索引更新：Z 个条目已同步
```

### 2. 新兴主题发现

**执行步骤：**

1. **分析知识覆盖度**
   - 按领域统计文章数量
   - 识别覆盖薄弱的领域（如强化学习仅有 1 篇）
   - 对比 AI 领域最新发展趋势

2. **识别新兴主题**
   - 关注以下方向的空白：
     - 多模态学习（Vision-Language Models）
     - 生成式 AI（Diffusion Models、GANs 进阶）
     - AI 安全与对齐（Alignment、RLHF）
     - 大模型推理优化（量化、蒸馏、加速）
     - AI 工程实践（MLOps、模型部署）
     - 图神经网络（GNN）
     - 自监督学习（Self-Supervised Learning）

3. **生成主题建议**
   - 为每个建议主题提供：标题、领域、难度、标签、简介
   - 按优先级排序（新兴领域 > 基础补充 > 进阶深化）

**输出格式：**
```
💡 新兴主题建议

高优先级：
1. 多模态学习基础
   - 领域：深度学习/计算机视觉
   - 难度：advanced
   - 标签：[multimodal, vision-language, clip]
   - 理由：当前知识体系缺少视觉-语言融合内容

2. AI安全与对齐
   - 领域：AI智能体
   - 难度：advanced
   - 标签：[alignment, safety, rlhf]
   - 理由：大模型时代的关键议题

中优先级：
3. 扩散模型原理
   - 领域：深度学习/生成式AI
   - 难度：advanced
   - 标签：[diffusion, generative, image-generation]
   - 理由：图像生成领域的主流技术
```

### 3. 内容质量优化

**执行步骤：**

1. **内容审查**
   - 检查现有文章是否符合质量标准
   - 识别需要更新的内容（如过时的技术、缺少最新进展）
   - 检查代码示例是否可运行

2. **内容增强**
   - 补充缺失的章节（如缺少"实际应用"或"思考问题"）
   - 更新技术细节（如添加最新的模型版本、性能指标）
   - 改进示例和案例

3. **一致性检查**
   - 确保术语使用统一
   - 检查交叉引用是否正确
   - 验证标签系统的一致性

**输出格式：**
```
🔧 内容优化报告

需要更新的文章：
1. llm/transformers.md
   - 问题：缺少 2023-2024 年最新进展
   - 建议：添加 Flash Attention、Longformer 等变体说明

2. deep-learning/neural-networks.md
   - 问题：缺少代码示例
   - 建议：添加 PyTorch/TensorFlow 实现示例

已优化：X 篇
- [x] 补充了 agents/agent-architecture.md 的实际应用案例
- [x] 更新了 nlp/overview.md 的大模型时代内容
```

### 4. 知识体系扩展

**执行步骤：**

1. **评估体系结构**
   - 检查当前目录结构是否合理
   - 识别需要新增的领域（如 "生成式AI"、"AI工程"）
   - 评估是否需要重组现有分类

2. **建议结构优化**
   - 提出目录结构调整方案
   - 说明调整理由和预期收益

3. **生成迁移计划**
   - 如需重组，提供详细的迁移步骤
   - 确保索引同步更新

**输出格式：**
```
🏗️ 知识体系扩展建议

建议新增领域：
1. generative-ai/ (生成式AI)
   - 理由：扩散模型、GANs、VAE 等生成技术需要独立分类
   - 建议文章：扩散模型原理、GANs进阶、VAE基础

2. ai-engineering/ (AI工程)
   - 理由：模型部署、MLOps、性能优化是实践关键
   - 建议文章：模型量化、推理加速、MLOps基础

结构调整：
- 当前 reinforcement-learning/ 仅有 1 篇文章
- 建议合并到 machine-learning/ 或扩充内容
```

### 5. 自动化进化流程

**执行步骤：**

1. **全量扫描**
   - 执行知识缺口识别
   - 执行新兴主题发现
   - 执行内容质量审查

2. **生成进化计划**
   - 按优先级排序所有待办事项
   - 提供预计工作量评估
   - 建议执行顺序

3. **执行进化**
   - 按优先级逐步执行补全、优化、扩展
   - 每完成一项更新进度
   - 最终生成进化报告

**输出格式：**
```
🚀 知识库进化计划

阶段 1：基础补全（高优先级）
- [ ] 补全 10 篇缺失文章
- [ ] 更新 3 篇过时内容

阶段 2：体系扩展（中优先级）
- [ ] 新增 generative-ai/ 领域
- [ ] 添加 5 篇新兴主题文章

阶段 3：质量优化（低优先级）
- [ ] 补充代码示例
- [ ] 统一术语表述

预计新增文章：15 篇
预计优化文章：8 篇
```

## 执行模式

### 模式 1：健康检查（默认）

仅执行扫描和诊断，生成报告但不修改文件。

**命令：** `bootstrap check`

### 模式 2：自动补全

识别缺失文章并自动创建。

**命令：** `bootstrap complete`

### 模式 3：主题发现

分析知识覆盖度，建议新兴主题。

**命令：** `bootstrap discover`

### 模式 4：全量进化

执行完整的知识库进化流程。

**命令：** `bootstrap evolve`

## 约束与规则

1. **内容质量标准**
   - 每篇文章必须包含：定义、核心概念、实际应用、典型例子、思考问题
   - 技术术语需有清晰解释
   - 代码示例需实际可运行（如提供）

2. **元数据规范**
   - 每篇文章必须标注：难度等级、标签、更新日期
   - 难度等级：beginner、intermediate、advanced
   - 标签：使用统一标签体系

3. **索引一致性**
   - 所有文章必须在 `topics-index.json` 中有对应条目
   - 索引中的 `last_updated` 必须保持最新

4. **知识覆盖均衡**
   - 确保各子领域都有代表性文章
   - 新兴领域优先更新
   - 基础领域定期维护

5. **版本控制**
   - 每次进化后提交更改
   - 提交信息格式：`docs: bootstrap进化 - [具体变更]`

- 索引与文件 100% 一致
- 每个领域至少有 2-3 篇代表性文章
- 所有文章符合质量标准（包含必需章节）
- 知识体系覆盖 AI 主要子领域
- 新兴主题及时纳入
