# 自然语言处理概述

## 定义

**自然语言处理（Natural Language Processing，NLP）** 是人工智能与语言学的交叉领域，旨在让计算机理解、生成和处理人类语言。

## NLP任务体系

### 基础任务
| 任务 | 说明 | 示例 |
|------|------|------|
| 分词 | 将文本切分成词 | "我爱中国" → ["我","爱","中国"] |
| 词性标注 | 标注词语词性 | "学习" → 动词 |
| 命名实体识别 | 识别人名地名等 | "北京" → 地点 |
| 依存句法分析 | 分析词语关系 | 主谓宾关系 |

### 高级任务
| 任务 | 说明 |
|------|------|
| 情感分析 | 判断文本情感倾向 |
| 机器翻译 | 跨语言转换 |
| 文本摘要 | 生成简短摘要 |
| 问答系统 | 回答用户问题 |
| 对话系统 | 人机对话交互 |

## 技术发展历程

```
2013 Word2Vec    → 词向量表示
2014 Seq2Seq    → 序列到序列模型
2015 Attention  → 注意力机制
2017 Transformer→ 革命性架构
2018 BERT/ GPT  → 预训练模型时代
2020 GPT-3      → 超大模型
2022 ChatGPT    → 对话式AI突破
2023 LLM爆发    → 多模态、长上下文
```

## 核心概念

### 词向量
将词语映射到稠密向量空间：
```
词 → 数值向量
king - man + woman ≈ queen
```

### Attention注意力机制
让模型关注输入的关键部分：
```
Query: 我想买一部
Key: 手机, 电脑, 书, 手机
Attention: 手机(0.8), 电脑(0.1)...
```

### Transformer架构
完全基于自注意力的序列建模：
- 并行计算效率高
- 捕获长距离依赖
- 可扩展性强

## 预训练模型时代

### BERT系列
- **BERT**：双向编码器，理解为主
- **RoBERTa**：优化训练策略
- **DeBERTa**：解耦注意力

### GPT系列
- **GPT-2**：文本生成
- **GPT-3/4**：few-shot学习
- **ChatGPT**：对话优化

### 开源模型
- **LLaMA**：Meta开源
- **ChatGLM**：清华开源
- **Qwen**：阿里开源
- **Mistral**：高性能

## 大模型时代的新范式

### Prompt Learning
通过提示词完成任务，无需fine-tuning

### RAG（检索增强生成）
结合知识库提升生成质量

### Agent
LLM作为大脑，调用工具完成复杂任务
