# 计算机视觉概述

## 定义

**计算机视觉（Computer Vision）** 是人工智能的一个重要分支，旨在让计算机能够"看懂"图像和视频，模拟人类视觉系统的感知能力。

## 核心任务

| 任务 | 描述 | 应用场景 |
|------|------|----------|
| 图像分类 | 识别图像中的物体类别 | 物品识别 |
| 目标检测 | 定位并识别图像中的物体 | 人脸检测 |
| 语义分割 | 对每个像素进行分类 | 自动驾驶道路分割 |
| 实例分割 | 区分同类不同个体 | 粒子计数 |
| 姿态估计 | 识别人体关键点 | 动作捕捉 |
| 图像生成 | 创建新图像 | AI绘图 |

## 发展历程

```
2012 AlexNet    → 深度学习崛起，CNN突破
2014 VGG/GoogLeNet → 网络结构深化
2015 ResNet    → 残差连接解决深度问题
2017 YOLO      → 实时目标检测
2020 Vision Transformer → Transformer应用于CV
2021 CLIP/SAM → 多模态和任意物体分割
```

## 经典模型

### CNN架构演进

```
LeNet(1998) → AlexNet(2012) → VGG(2014) → ResNet(2015) → EfficientNet(2019)
```

### Vision Transformer (ViT)

将Transformer架构应用于图像：
```
图像 → 切分patches → 线性嵌入 → Transformer编码器 → 分类
```

## 核心概念

### 卷积操作
- 滤波器（卷积核）在图像上滑动
- 提取边缘、纹理等特征
- 权重复用，减少参数量

### 特征金字塔（FPN）
- 多尺度特征融合
- 检测不同大小的物体

### 非极大值抑制（NMS）
- 去除重复检测框
- 保留最优检测结果

## 应用领域

### 日常生活
- 手机人脸解锁
- 拍照HDR和美颜
- 扫码支付
- AR滤镜

### 行业应用
- **医疗**：X光片分析、癌细胞检测
- **自动驾驶**：障碍物检测、车道保持
- **安防**：人群监控、异常行为检测
- **制造**：产品缺陷检测
- **农业**：作物病虫害识别

## 主流数据集

| 数据集 | 规模 | 用途 |
|--------|------|------|
| ImageNet | 1400万+图像 | 预训练/分类 |
| COCO | 33万图像 | 检测/分割 |
| PASCAL VOC | 1万+图像 | 多任务 |
| CelebA | 20万+人脸 | 人脸识别 |

## 评价指标

- **Top-1/Top-5准确率**：分类性能
- **mAP（mean Average Precision）**：检测性能
- **IoU（Intersection over Union）**：分割精度
