# 知识库

> AI 相关领域知识整理

## 五大板块

本知识库按照「随手记 → 学科历史 → 基础知识 → 工程应用 → 前沿探索」的认知递进路径组织。

| 板块 | 目录 | 说明 |
|------|------|------|
| 随手记 | `notes/` | 灵感想法、代码片段、问题思考、日常记录 |
| 学科历史 | `history/` | AI 发展时间线、里程碑、人物传记、流派演变 |
| 基础知识 | `fundamentals/` | 核心概念、数学基础、ML、DL、NLP、CV |
| 工程应用 | `applications/` | 实战项目、工具使用、代码范例、最佳实践 |
| 前沿探索 | `frontier/` | 论文笔记、最新研究、新兴技术、趋势分析 |

## 目录结构

```
knowledge-base/
├── notes/                 # 随手记
│   ├── ideas/            # 灵感想法
│   ├── snippets/         # 代码片段
│   ├── questions/        # 问题思考
│   └── daily/            # 日常记录
├── history/              # 学科历史
│   ├── timeline/         # 发展时间线
│   ├── milestones/       # 里程碑事件
│   ├── pioneers/        # 人物传记
│   └── schools/          # 流派演变
├── fundamentals/         # 基础知识
│   ├── concepts/         # 核心概念
│   ├── math/            # 数学基础
│   ├── ml/              # 机器学习
│   ├── dl/              # 深度学习
│   ├── nlp/             # NLP 基础
│   └── cv/              # CV 基础
├── applications/         # 工程应用
│   ├── projects/         # 实战项目
│   ├── tools/           # 工具使用
│   ├── code/            # 代码范例
│   ├── practices/       # 最佳实践
│   └── tutorials/       # 教程指南
├── frontier/             # 前沿探索
│   ├── papers/           # 论文笔记
│   ├── research/        # 研究动态
│   ├── emerging/         # 新兴技术
│   └── trends/          # 趋势分析
├── templates/            # 笔记模板
│   ├── quick-note-template.md
│   ├── book-note-template.md

```
