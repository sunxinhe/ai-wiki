# 机器学习概述

## 定义

**机器学习（Machine Learning，ML）** 是人工智能的一个子领域，专注于开发能够从数据中自动学习和改进的算法。不同于传统编程需要显式编写规则，机器学习让计算机自动从数据中发现模式并做出预测或决策。

## 机器学习 vs 传统编程

```
传统编程：规则 + 数据 → 程序
机器学习：数据 + 答案 → 程序
```

## 机器学习的三种主要类型

### 1. 监督学习（Supervised Learning）

**定义**：从标注数据中学习，输入与期望输出配对。

**典型任务**：
- 分类（Classification）：判断样本类别
- 回归（Regression）：预测连续数值

**常见算法**：
- 线性回归、逻辑回归
- 决策树、随机森林
- 支持向量机（SVM）
- 神经网络

### 2. 无监督学习（Unsupervised Learning）

**定义**：从无标注数据中发现模式。

**典型任务**：
- 聚类（Clustering）：将相似样本分组
- 降维（Dimensionality Reduction）：减少特征数量
- 异常检测（Anomaly Detection）：发现异常数据

**常见算法**：
- K-Means
- 层次聚类
- PCA
- GAN（生成对抗网络）

### 3. 强化学习（Reinforcement Learning）

**定义**：通过与环境交互学习最优策略。

**核心概念**：
- 智能体（Agent）
- 环境（Environment）
- 状态（State）
- 动作（Action）
- 奖励（Reward）

## 机器学习工作流程

```
1. 数据收集 → 2. 数据预处理 → 3. 特征工程 → 4. 模型选择 → 5. 训练模型 → 6. 评估优化 → 7. 部署
```

## 重要术语

| 术语 | 解释 |
|------|------|
| 特征（Feature） | 输入数据的属性 |
| 标签（Label） | 期望的输出结果 |
| 训练集 | 用于训练模型的数据 |
| 测试集 | 用于评估模型的数据 |
| 过拟合 | 模型在训练集表现好，测试集差 |
| 欠拟合 | 模型在训练集表现差 |

## 实际应用

- **推荐系统**：抖音/淘宝的商品推荐
- **垃圾邮件检测**：自动识别垃圾邮件
- **房价预测**：基于特征的房价评估
- **图像识别**：人脸识别、物体检测
